BLT Design Showcases

Community-driven design submissions for OWASP BLT. Browse entries, react with 👍 on GitHub, and submit your own work.

How it works

  1. 1

    Submit via GitHub

    Open a new issue using the Design Submission template. Upload your preview image, add a description and a link to your design.

  2. 2

    Community rates it

    Anyone can leave a 👍 reaction on your issue to show appreciation. The showcase automatically reflects the current reaction counts.

  3. 3

    Showcase updates

    GitHub Actions rebuilds this page whenever a submission issue is opened or edited, keeping the showcase always up to date.
